# 🧠 Memora – A Personalized Dementia Care Companion

**Memora** is a subscription-based mobile and web application designed to support **dementia patients, caregivers, families, and volunteers**.

It provides personalized care management, real-time monitoring, and community-driven support to improve the lives of dementia patients and reduce the burden on caregivers.

---

## 🌟 Features

### 👩‍⚕️ Patients & Caregivers

- Add **daily tasks**: activities, medications, appointments
- **Predefined essential activities** based on dementia stages (including **games for cognitive well-being**)
- **Timely notifications** to ensure no tasks are missed
- **Daily activity & task reports** accessible by guardians/caregivers
- **Task assignment rules**:
  - **Mild stage patients**: both the patient and caregiver can add tasks
  - **Moderate to severe stages**: only the caregiver can manage and assign tasks

### 💳 Subscription Model

- Guardians register patients on the platform
- Access to all care features requires an **active subscription plan**
- Subscription ensures **sustainability of the platform** and ongoing **feature improvements**

### ⌚ Smartwatch Integration

- Real-time monitoring of patients' activity and health
- Detection of **falls, anomalies, or accidents**
- **Instant alerts** sent to caregivers/guardians for quick action

### 🤝 Volunteers

- Volunteers can **only access the web dashboard**
- Features:
  - Register and manage volunteer profile (requires **admin approval**)
  - **Add educational content** (articles, tips, resources)
  - Manage and publish content for patients/caregivers

### 🛠️ Web Dashboard (Admin + Volunteers)

**Admins**:
- Approve volunteers during registration
- Monitor platform activity and manage users
- Generate **reports and analytics** for insights
- Approve volunteer-created content before publishing

**Volunteers**:
- Create and manage educational content

## 🏗️ Tech Stack

- **Mobile App**: [Flutter](https://flutter.dev/)

- **Relational Database**: [PostgreSQL](https://www.postgresql.org/)

## 💼 Business Model

Memora follows a **subscription-based model**:

- Guardians pay a monthly/annual fee when registering patients
- Subscription unlocks access to premium features such as:
  - Personalized daily task scheduling
  - Stage-based predefined activities & cognitive games
  - Real-time smartwatch monitoring
  - Detailed patient reports & analytics
  - Access to volunteer educational content
- Ensures long-term **sustainability** and **continuous platform improvements**
